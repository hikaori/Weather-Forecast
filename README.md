
## Weather-Forecast

Get weather forecast for a given city.


### API
https://openweathermap.org/api

### Librarys
classnames
styled-components
react-autosuggest
react-redux


### How to start
`yarn start`


### note
if the API key expired you can get key from 
<a href="https://openweathermap.org/api">here</a>
